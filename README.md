# WR Tasks Frontend

SPA для управления задачами на Vue 3 + TypeScript + Vite.

## Требования

- Node.js 20+
- Доступный backend API

## Переменные окружения

Скопируйте `.env.example` в `.env` и укажите URL API:

`VITE_API_BASE_URL=http://localhost:8080`

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
npm run preview
```
